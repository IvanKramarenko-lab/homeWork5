p {} - Селектор по типу элемента.
#id {} - Селектор по идентификатору.
.class {} - Селектор по классу.
[attr] {} - Селектор по атрибуту.
A, B {} - Комбинатор запятая.
A B {} - Комбинатор потомков.
A > B {} - Дочерние селекторы.
A ~ B {} - Комбинатор всех соседних элементов.
A + B {} - Комбинатор следующего соседнего элемента.
